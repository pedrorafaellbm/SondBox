# 🎵 SondBox

SondBox é uma plataforma gratuita de streaming de músicas, podcasts, rádios e conteúdo em áudio, desenvolvida como Progressive Web App (PWA).

O projeto tem como objetivo oferecer uma experiência comparável aos principais serviços de streaming, porém totalmente gratuita, moderna, rápida e altamente personalizável.
