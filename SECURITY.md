# Security

## Como reportar vulnerabilidades
Reporte vulnerabilidades via issue no GitHub ou envie um e-mail para a equipe de segurança do projeto.

## Política de segurança
- Não publique vulnerabilidades publicamente antes de serem tratadas.
- Reporte o problema de forma responsável.
- Priorize correção e mitigação antes de divulgação.

## Versões suportadas
Suportar a versão atual e as duas versões anteriores enquanto possível.
